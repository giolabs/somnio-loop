# somnio-loop

**Ticket adentro. Entregable afuera.**

somnio-loop es un plugin de Claude Code que ejecuta un loop agéntico completo de punta a punta sin intervención adicional. Dale un ticket — un prompt en texto libre, un ID de Jira/Linear/GitHub Issues, o un párrafo pegado directamente — y se encarga automáticamente del triage, el fan-out paralelo de sub-agentes, la verificación auto-sanante, la generación de specs y el registro de ADRs. Es agnóstico a la tecnología y funciona con cualquier stack que tenga un manifest detectable.

Para el release actual, consultá el [CHANGELOG](https://github.com/giosomniodev/somnio-loop/blob/main/CHANGELOG.md).
